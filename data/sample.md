# Documento de ejemplo

Este repositorio demuestra un flujo RAG mínimo con LangChain:
- Carga de documentos
- Fragmentación por caracteres
- Embeddings con OpenAI
- Almacenamiento vectorial FAISS
- Recuperación + generación con un LLM

La pregunta "¿De qué trata el archivo de ejemplo?" debería devolver un resumen como respuesta.
